## TypeScript

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **TypeScript** | The compiler and language for backend code (`strict` mode). | [typescriptlang.org/docs](https://www.typescriptlang.org/docs/) |
| **TSConfig Paths** | Resolves `tsconfig` path aliases at runtime. | [github.com/dividab/tsconfig-paths](https://github.com/dividab/tsconfig-paths) |
| **Source Map Support** | Maps runtime stack traces back to TypeScript source. | [github.com/evanw/node-source-map-support](https://github.com/evanw/node-source-map-support) |
