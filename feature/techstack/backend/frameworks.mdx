## Node.js Frameworks — NestJS Ecosystem

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **NestJS Core** | Progressive Node.js framework with a modular, DI-based architecture. | [docs.nestjs.com](https://docs.nestjs.com) |
| **NestJS Common** | Shared decorators, pipes, guards, and interceptors. | [docs.nestjs.com/first-steps](https://docs.nestjs.com/first-steps) |
| **NestJS Config** | Typed configuration and environment management. | [docs.nestjs.com/techniques/configuration](https://docs.nestjs.com/techniques/configuration) |
| **NestJS JWT** | JWT signing and verification module. | [docs.nestjs.com/security/authentication](https://docs.nestjs.com/security/authentication) |
| **NestJS Passport** | Passport integration for authentication strategies. | [docs.nestjs.com/recipes/passport](https://docs.nestjs.com/recipes/passport) |
| **NestJS Swagger** | Auto-generates OpenAPI docs from decorators. | [docs.nestjs.com/openapi/introduction](https://docs.nestjs.com/openapi/introduction) |
| **NestJS Testing** | Utilities for unit and e2e testing Nest modules. | [docs.nestjs.com/fundamentals/testing](https://docs.nestjs.com/fundamentals/testing) |
| **NestJS CLI** | Scaffolding and build tooling for Nest projects. | [docs.nestjs.com/cli/overview](https://docs.nestjs.com/cli/overview) |
| **NestJS Schematics** | Code generation schematics used by the CLI. | [github.com/nestjs/schematics](https://github.com/nestjs/schematics) |
