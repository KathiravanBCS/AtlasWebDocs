## ORM — Prisma Ecosystem

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Prisma ORM** | Type-safe ORM with schema-driven migrations. | [prisma.io/docs](https://www.prisma.io/docs) |
| **Prisma Client** | The generated, type-safe query client. | [prisma.io/docs/orm/prisma-client](https://www.prisma.io/docs/orm/prisma-client) |
