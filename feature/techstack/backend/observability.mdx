## Logging — Pino Ecosystem

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Pino** | Very low-overhead structured JSON logger for Node. | [getpino.io](https://getpino.io) |
| **Pino HTTP** | HTTP request logging middleware built on Pino. | [github.com/pinojs/pino-http](https://github.com/pinojs/pino-http) |
| **Pino Pretty** | Human-readable log formatting for local development. | [github.com/pinojs/pino-pretty](https://github.com/pinojs/pino-pretty) |
| **NestJS Pino** | Pino integration for Nest with request-scoped logging. | [github.com/iamolegga/nestjs-pino](https://github.com/iamolegga/nestjs-pino) |
