## Enterprise architecture summary

The stack rolls up into these architectural roles:

| Area              | Technologies                                            |
| ----------------- | ------------------------------------------------------- |
| UI frameworks     | React · Next.js                                         |
| Design systems    | Mantine · Radix UI                                      |
| Styling           | Tailwind CSS · Vanilla Extract · PostCSS                |
| State management  | TanStack Query · Zustand                                |
| Authentication    | Azure MSAL                                              |
| Authorization     | CASL                                                    |
| Tables            | AG Grid · TanStack Table · Mantine DataTable            |
| Editors           | RoosterJS · TipTap                                      |
| Charts            | Recharts · D3 · Chart.js · Cytoscape                    |
| AI UI             | Assistant UI · AG UI · LangGraph                        |
| Documents         | PDF · Excel · CSV                                       |
| Testing           | Vitest · Testing Library · Storybook                    |
| Utilities         | Lodash · UUID · Zod · Day.js · date-fns                 |
