## CSS-in-TS — Vanilla Extract

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Vanilla Extract CSS** | Type-safe, zero-runtime CSS written in TypeScript. | [vanilla-extract.style](https://vanilla-extract.style) |
| **Vanilla Extract Vite Plugin** | Compiles `.css.ts` files during the Vite build. | [vanilla-extract.style/documentation/integrations/vite](https://vanilla-extract.style/documentation/integrations/vite/) |

## Tailwind Ecosystem — Core

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Tailwind CSS 4** | Utility-first CSS framework with the new high-performance engine. | [tailwindcss.com/docs](https://tailwindcss.com/docs) |

## Tailwind Plugins

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Tailwind Animate** | Utility classes for enter/exit animations. | [github.com/jamiebuilds/tailwindcss-animate](https://github.com/jamiebuilds/tailwindcss-animate) |
| **Tailwind Scrollbar** | Utilities for styling scrollbars. | [github.com/adoxography/tailwind-scrollbar](https://github.com/adoxography/tailwind-scrollbar) |
| **Tailwind Merge** | Intelligently merges conflicting Tailwind classes. | [github.com/dcastil/tailwind-merge](https://github.com/dcastil/tailwind-merge) |

## PostCSS

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **PostCSS** | Tool for transforming CSS with JS plugins. | [postcss.org](https://postcss.org) |
| **PostCSS Preset Mantine** | PostCSS functions and mixins used by Mantine. | [mantine.dev/styles/postcss-preset](https://mantine.dev/styles/postcss-preset/) |
| **PostCSS Simple Vars** | Sass-like variables in PostCSS. | [github.com/postcss/postcss-simple-vars](https://github.com/postcss/postcss-simple-vars) |
| **Tailwind PostCSS Plugin** | Runs Tailwind as a PostCSS plugin. | [tailwindcss.com/docs/installation/using-postcss](https://tailwindcss.com/docs/installation/using-postcss) |
| **Autoprefixer** | Adds vendor prefixes automatically. | [github.com/postcss/autoprefixer](https://github.com/postcss/autoprefixer) |
