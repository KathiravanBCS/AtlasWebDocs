## Markdown Rendering

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **React Markdown** | Renders Markdown to React elements safely (no `dangerouslySetInnerHTML`). | [github.com/remarkjs/react-markdown](https://github.com/remarkjs/react-markdown) |
| **Remark GFM** | GitHub Flavored Markdown — tables, strikethrough, task lists, autolinks. | [github.com/remarkjs/remark-gfm](https://github.com/remarkjs/remark-gfm) |

## Math Support

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Remark Math** | Parses `$...$` math syntax in Markdown. | [github.com/remarkjs/remark-math](https://github.com/remarkjs/remark-math) |
| **Rehype KaTeX** | Renders parsed math with KaTeX. | [github.com/remarkjs/remark-math/tree/main/packages/rehype-katex](https://github.com/remarkjs/remark-math/tree/main/packages/rehype-katex) |
| **KaTeX** | Fast math typesetting library for the web. | [katex.org](https://katex.org) |
