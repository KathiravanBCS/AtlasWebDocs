## Unit Testing — Vitest

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Vitest** | Fast Vite-native unit/integration test runner with a Jest-compatible API. | [vitest.dev](https://vitest.dev) |
| **Testing Library Jest DOM** | Custom matchers like `toBeInTheDocument()` for DOM assertions. | [github.com/testing-library/jest-dom](https://github.com/testing-library/jest-dom) |

## Component Testing — Testing Library

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **React Testing Library** | Test components the way users interact with them. | [testing-library.com/react](https://testing-library.com/docs/react-testing-library/intro/) |
| **DOM Testing Library** | The DOM-querying core behind the framework wrappers. | [testing-library.com/dom](https://testing-library.com/docs/dom-testing-library/intro/) |
| **User Event** | Simulates realistic user interactions (typing, clicking). | [testing-library.com/user-event](https://testing-library.com/docs/user-event/intro/) |
