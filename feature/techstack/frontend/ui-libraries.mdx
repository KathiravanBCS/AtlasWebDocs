## Mantine Ecosystem — Core

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Mantine Core** | The primary component library — inputs, layout, overlays, and more. | [mantine.dev/getting-started](https://mantine.dev/getting-started/) |
| **Mantine Hooks** | A large collection of reusable React hooks. | [mantine.dev/hooks/use-disclosure](https://mantine.dev/hooks/use-disclosure/) |
| **Mantine Form** | Form state, validation, and field management. | [mantine.dev/form/use-form](https://mantine.dev/form/use-form/) |
| **Mantine Dates** | Date and calendar pickers. | [mantine.dev/dates/getting-started](https://mantine.dev/dates/getting-started/) |
| **Mantine Notifications** | Toast notification system. | [mantine.dev/x/notifications](https://mantine.dev/x/notifications/) |
| **Mantine Modals** | Centralized modal manager (confirm, content modals). | [mantine.dev/x/modals](https://mantine.dev/x/modals/) |

## Mantine Ecosystem — Additional Packages

| Technology | Description | Study link |

## Radix UI — Components

Unstyled, accessible primitives we style ourselves.

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Radix Avatar** | Accessible avatar with image fallback. | [radix-ui.com/primitives/docs/components/avatar](https://www.radix-ui.com/primitives/docs/components/avatar) |
| **Radix Dialog** | Accessible modal dialog with focus management. | [radix-ui.com/primitives/docs/components/dialog](https://www.radix-ui.com/primitives/docs/components/dialog) |
| **Radix Label** | Accessible form label primitive. | [radix-ui.com/primitives/docs/components/label](https://www.radix-ui.com/primitives/docs/components/label) |
| **Radix Separator** | Visual or semantic separator. | [radix-ui.com/primitives/docs/components/separator](https://www.radix-ui.com/primitives/docs/components/separator) |
| **Radix Slot** | Merges props onto a child element (`asChild` pattern). | [radix-ui.com/primitives/docs/utilities/slot](https://www.radix-ui.com/primitives/docs/utilities/slot) |
| **Radix Switch** | Accessible toggle switch. | [radix-ui.com/primitives/docs/components/switch](https://www.radix-ui.com/primitives/docs/components/switch) |
| **Radix Tooltip** | Accessible hover/focus tooltip. | [radix-ui.com/primitives/docs/components/tooltip](https://www.radix-ui.com/primitives/docs/components/tooltip) |
