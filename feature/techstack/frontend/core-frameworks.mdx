The base layer everything else builds on.

## React Ecosystem

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **React 19** | The component-based UI library at the core of the stack, with Server Components, Actions, and the new compiler. | [react.dev](https://react.dev) |
| **React DOM** | The renderer that mounts React components into the browser DOM. | [react.dev/reference/react-dom](https://react.dev/reference/react-dom) |
| **React Router DOM** | Client-side routing for React apps not using Next.js's router. | [reactrouter.com](https://reactrouter.com) |

## Next.js Ecosystem

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Next.js 15** | The React framework — App Router, server actions, rendering, and bundling. | [nextjs.org/docs](https://nextjs.org/docs) |
| **Next Themes** | Dark/light theme management for Next.js with no flash on load. | [github.com/pacocoursey/next-themes](https://github.com/pacocoursey/next-themes) |
