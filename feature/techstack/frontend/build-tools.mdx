The tooling that compiles, bundles, and serves the app during development and for production.

## Build Tools

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Vite** | Fast dev server and build tool with instant HMR. | [vite.dev](https://vite.dev) |
| **TurboRepo** | High-performance monorepo build system with caching. | [turborepo.com/docs](https://turborepo.com/docs) |
| **ESBuild** | Extremely fast JS/TS bundler and minifier used under the hood. | [esbuild.github.io](https://esbuild.github.io) |

## Vite Plugins

| Technology | Description | Study link |
| ---------- | ----------- | ---------- |
| **Vite React Plugin** | Enables React Fast Refresh and JSX in Vite. | [github.com/vitejs/vite-plugin-react](https://github.com/vitejs/vite-plugin-react) |
| **Vite TSConfig Paths** | Resolves TypeScript `paths` aliases in Vite. | [github.com/aleclarson/vite-tsconfig-paths](https://github.com/aleclarson/vite-tsconfig-paths) |
| **Vanilla Extract Vite Plugin** | Compiles Vanilla Extract `.css.ts` files in Vite. | [vanilla-extract.style/documentation/integrations/vite](https://vanilla-extract.style/documentation/integrations/vite/) |
| **Vite CSS Modules Plugin** | CSS Modules support and configuration in Vite. | [vite.dev/guide/features#css-modules](https://vite.dev/guide/features#css-modules) |
| **Vite Bundle Analyzer** | Visualizes bundle composition to find size regressions. | [github.com/btd/rollup-plugin-visualizer](https://github.com/btd/rollup-plugin-visualizer) |
| **Rollup Visualizer** | Generates a treemap of the production bundle. | [github.com/btd/rollup-plugin-visualizer](https://github.com/btd/rollup-plugin-visualizer) |
| **MKCert Plugin** | Generates locally-trusted HTTPS certs for the dev server. | [github.com/liuweiGL/vite-plugin-mkcert](https://github.com/liuweiGL/vite-plugin-mkcert) |
