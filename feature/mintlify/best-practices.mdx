## Pre-launch checklist

- [ ] `docs.json` validates and includes `$schema`
- [ ] Light/dark logos and favicon set
- [ ] Every page reachable from `navigation` (no orphans)
- [ ] OpenAPI spec passes `mint openapi-check`
- [ ] Custom domain configured
- [ ] `mint broken-links` runs clean in CI
- [ ] Redirects added for any changed URLs
- [ ] SEO `description` written on every page
- [ ] Analytics integration connected
- [ ] AI assistant and contextual menu enabled
- [ ] SSO / access control configured (if internal)
