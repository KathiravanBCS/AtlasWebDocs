| Symptom                                   | Likely cause                                  | Fix                                                       |
| ----------------------------------------- | --------------------------------------------- | -------------------------------------------------------- |
| Page renders but isn't in the sidebar     | Not listed in `navigation`                    | Add its path to a `group` in `docs.json`.                |
| Local dev shows stale or broken content   | Cached build artifacts                        | Delete `.mintlify` and rerun `mint dev`.                 |
| API reference looks incomplete            | Invalid or partial OpenAPI spec               | Run `mint openapi-check`; fix the spec, not the page.    |
| Deploy didn't happen after merge          | Build failed on a broken page or config       | Check the build log; fix MDX/`docs.json`, push again.    |
| Old links 404 after a restructure         | URLs changed without redirects                | Add entries to `redirects` in `docs.json`.               |
| Search returns nothing for real terms     | Thin or poorly-headed content                 | Add real headings and self-contained sections.           |
