<Info>
  This is a working reference, not a marketing tour. It assumes you can read JSON
  and run a terminal, and it favors the *current* Mintlify stack: the `mint` CLI
  and the `docs.json` configuration format (which replaced the older `mintlify`
  CLI and `mint.json`).
</Info>

## How to use this guide

Read it in order the first time to build a mental model, then treat it as a
lookup table. Each page is self-contained and ends with the gotchas that
actually bite people in production.
